CSS reset
设置 HTML 标签的默认样式，使其在各个浏览器表现基本一致，让默认样式归零。
常见设置有 marging、padding、inherit、outline 等。

normalize.css
设置 HTML 标签的默认样式，使其在各个浏览器表现基本一致，保留标签的默认样式。

CSS 模块原则
面向对象（OOCSS）

开闭原则
Don’t Repeat Yourself（DRY）

面向对象（OOCSS）
把基础的样式抽象出来，不一样的地方做继承。

.btn {
    display: inline-block;
    padding: 0.36em 0.8em;
    margin-right: 0.5em;
    border: none;
    border-radius: 0.3em;
    cursor: pointer;
}
.btn--primary {
    background-color: #09c;
    color: #fff;
}

OOCSS 原则
结构和皮肤分离：.bn .btn–primary .btn–info .btn–danger
容器和内容分离：.header .btn { color: #fff; } /* 不推荐这样写 */

media object

.media
    .media__image
        .media_object
    .media__body
        .media__title
        .media__description

单一职责原则
尽可能细地拆分成可独立复用的组件，通过组合方式使用多个组件。

开闭原则
对扩展开放，对修改关闭。

.box { display: block; padding: 10px; }
.content .box { padding: 20px; } /* bad example */
.box--large { padding: 20px; } /* better example */

DRY 原则

/* bad example */
.selector {
    border-top: 1px solid #fcc;
    border-left: 1px solid #fcc;
    border-bottom: 1px solid #fcc;
}
/* better example */
.selector {
    border: 1px solid #fcc;
    border-right: none;
}

命名
基于功能：.button/.form/.list/.external-link/.tab-item/（优先使用）
基于内容：.news/.user-info/.help/.contact-me（适合中小型网站）
基于视觉：.round-image/.nowrap（适合大型网站）
建议使用抽象的命名：
.warning vs .orange
.btn-primary vs .btn-blue
.size-large vs .width-200

BEM
CSS 命名规范的一种：Block（模块）、Element（元素）、Modifier（修饰）

    <a class="tabs__item" href="#">Home</a>
    <a class="tabs__item" href="#">JavaScript</a>
    <a class="tabs__item" href="#">CSS</a>
</nav>

CSS 预处理
less, Sass, Stylus, PostCSS（CSS 界的 babel）

CSS 笔记：兼容性

Posted on 2017-08-18

兼容性问题
浏览器不支持该特性。
某些特定条件下出发浏览器 bug。

解决方案

  • 低版本浏览器没有这个特性可以接受吗？
    border-radius 不支持时，没有圆角。
    box-shadow 不支持时，没有阴影。
  • 可以使用效果稍微差一些的替代方案吗？
    min-height: 100vh，用 min-height: 800px。
  • 可以使用一些特带方案吗？
    opacity: 0.5 在 IE 下用 filter: alpha(opacity=50)。
  • 可以使用 JavaScript 让浏览器支持吗？
    使用 html5shim.js 让 IE6~8 支持新标签。
    使用 DD_belatedPNG.js 让IE6 支持半透明 png 图片。
  • 更换实现方式。

浏览器支持情况
caniuse.com
MDN
Codrops CSS Reference
QuirksMode.org - CSS

浏览器市场份额
日志分析
百度统计、NetMarketShare

浏览器前缀
浏览器产商为了实验新特性，在属性名前加前缀。
Chrome/Safari/Opera: -webkit-
Microsoft: -ms-
Mozilla: -moz-

不同浏览器使用不同的样式
@supports、层叠、条件注释、浏览器怪癖

CSS3 属性
IE6-7 不支持
IE8 部分支持
IE9 基本都支持

IE6、7 不支持
IE6、7 很多都不支持

IE8 不支持
background-size：推荐在 IE8 及以下使用固定宽度布局。
border-radius
box-shadow
opacity：filter: alpha(opacity=50)
rgba、hsl、hsla：一般场景下，使用相近的不透明颜色代替。
rem/vh/vw/calc：降级为固定宽度。

IE9 不支持
transition 与 animation：可以接受降级，如果不能接受就用 JavaScript。

控制 IE 模式（<=10）
meta 标签控制进入哪种文档模式。

<!-- 使用IE7模式渲染-->
<meta http-equiv="X-UA-Compatible" content="IE=7">
<!-- 使用最新引擎-->
<meta http-equiv="x-ua-compatible" content="IE=edge">

兼容写法工具
Autoprefixer

html5shiv.js
让 IE 系列支持 html5 语义化标签。

测试兼容性
虚拟机
BrowserStack

Polyfill
使用代码帮助浏览器实现它尚未支持的特性。
box-sizing-polyfill
flexibility
selectivizr
CSS3 PIE
cssSandpaper

最后的建议
推荐使用新的写法，通过 Polyfill 和 Autoprefixer 等工具帮助处理。
不要因为兼容性而牺牲新特性，使用旧的写法。

CSS 笔记：布局

Posted on 2017-08-18

布局方法
float, position, inline-block, table, flex, grid

水平居中
行级元素：text-align: center
块级元素：margin: auto

max-width
大部分图文内容的网页，布局都可以用 max-width 加 margin: auto。

垂直居中
单行文字：line-height
行级盒子：vertical-align: middle
绝对定位：top: 50%; left: 50%

position
用 absolute 和 margin 进行两栏布局。

main { position: relative; }
article { margin-left: 10em; }
aside { position: absolute; width: 10em; top: 0; left: 0; }

table
用 display 模拟表格布局，宽度自适应。不要使用表格标签布局。

main { display: table; }
article { display: table-cell; }
aside { display: table-cell; }

可用于实现居中效果。
.container { display: table-cell; text-align: center; vertical-align: middle; }

display: flex
将元素变为 Flexbox。

flex-direction
子元素排列方向，取值为 row、row-reverse、column、column-reverse。

flex-grow
元素扩展（占剩余空间）的能力，默认值为 0。

flex-shrink
元素收缩的能力，默认值为 1。

flex-wrap
元素在主轴方向排放时，能否换行，取值为 nowrap、wrap、wrap-reverse。

justify-content
子元素沿主轴方向的摆放，取值为 flex-start、flex-end、center、space-between、space-around。

align-items
子元素在侧轴方向的对齐方式，取值为 flex-start、flex-end、center、baseline、stretch。

align-self
对单独的元素设置对齐方式。

align-content
多行内容在容器内侧轴方向的对齐，取值为 flex-start、flex-end、center、space-between、space-around、stretch。

order
指定摆放时的顺序，默认值为 0。

CSS 笔记：层叠和单位

Posted on 2017-08-18

属性覆盖
优先级高的属性覆盖优先级低的属性， 减少了代码量。

<button class="btn">普通按钮</button>
<button class="btn btn-primary">主要按钮</button>
<style>
    .btn {
        display: inline-block;
        padding: .36em .8em;
        margin-right: .5em;
        line-height: 1.5;
        text-align: center;
        cursor: pointer;
        border-radius: .3em;
        border: none;
        background: #e6e6e6;
        color: #333;
    }
    .btn.btn-primary {
        color: #fff;
        background: #218de6;
    }
</style>

相对单位
em：相对于该元素的 font-size
rem：相对于根元素（html）的 font-size
vh：浏览器窗口高度的1%
vw：浏览器窗口宽度的1%
vmin：vh 和 vw 中的较小者
vmax：vh 和 vw 中的较大者

颜色
关键字：blue, lightsteelblue(147种)
Hex：#0000ff
RGB & RGBA：rgb(0,0,255), rgba(0,0,0,0.5)
HSL & HSLA：hsl(0,50%,50%), hsla(300, 50%, 50%, 0.5)
HSL 用于形成同一色系，如按钮悬浮时加深颜色。
