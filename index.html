晶体管（有开、关两种稳定电信号） -> 1和0（true、false） -> 逻辑门（||、&&、!、^） -> 半加器（2个逻辑门计算1bit的加法） -> ALU（70个逻辑门加减运算） -> CPU

基本数据结构

Posted on 2019-02-16

资料来自github

链表
链表是数据元素的线性集合，元素的线性顺序不是由它们在内存中的物理位置给出的，相反，每个元素指向下一个元素。

队列
队列是一种特殊类型的抽象数据类型或集合，集合中的实体按顺序保存。
队列基本操作有两种：入队，向队列的后端位置添加实体；出队，并从队列的前端位置移除实体。
可以概括为“先进先出（FIFO = first in, first out）”

栈
栈是一种抽象数据类型，用作表示元素的集合。
栈基本操作有两种：push，添加元素到栈的顶端（末尾）；pop，移除栈最顶端（末尾）的元素。
可疑概括为“后进先出（LIFO = last in, first out）”

堆
堆是一种特殊的基于树的数据结构，它满足下面描述的堆属性：
最小堆：父节点的值小于或等于子节点的值。

最大堆：父节点的值大于或等于子节点的值。

基本排序算法（JavaScript）

Posted on 2019-02-16

资料来自掘金

冒泡排序
两两比较，把最大的放到后面

function bubleSort(arr) {
    var len = arr.length;
    for (let outer = len ; outer >= 2; outer--) {
        for(let inner = 0; inner <=outer - 1; inner++) {
            if(arr[inner] > arr[inner + 1]) {
                [arr[inner],arr[inner+1]] = [arr[inner+1],arr[inner]]
            }
        }
    }
    return arr;
}

选择排序
遍历元素，如果比第一个小，就交换位置

function selectSort(arr) {
    var len = arr.length;
    for(let i = 0 ;i < len - 1; i++) {
        for(let j = i ; j<len; j++) {
            if(arr[j] < arr[i]) {
                [arr[i],arr[j]] = [arr[j],arr[i]];
            }
        }
    }
    return arr
}

快速排序
选一个基准值，剩下小的放左边，大的放右边，递归左边和右边

function quickSort(arr) {
    if(arr.length <= 1) {
        return arr;  //递归出口
    }
    var left = [],
        right = [],
        current = arr.splice(0,1); 
    for(let i = 0; i < arr.length; i++) {
        if(arr[i] < current) {
            left.push(arr[i])  //放在左边
        } else {
            right.push(arr[i]) //放在右边
        }
    }
    return quickSort(left).concat(current,quickSort(right));
}

在小程序中使用字体图标

Posted on 2018-10-08

准备 svg 格式的图标
可以到各种图标网站下载，或者自己用 Adobe Illustrator 画。

使用 IcoMoon 制作字体
进入 IcoMoon 网站，IcoMoon App -> New Project(not necessary) -> Import Icons -> Generate Font -> Download，完成下载。

引入样式文件
将下载的 style.css 文件引入项目。

字体转 base64 格式（针对小程序）
准备好下载的 fonts/xxx.woff 字体文件，进入 trasfonter 网站，Add fonts -> .woff -> Base64 encode -> Convert -> Download，完成下载。

替换字体代码（针对小程序）
用下载的 stylesheet.css 文件代码，替换 style.css 文件中 @font-face 代码。并把 [class^="icon-"], [class*=" icon-"] 改为 .icon。

使用字体图标
以类似 <span class="icon icon-home"></span> 的方式使用字体图标，设置字体大小。

float 和 position:absolute 脱离文本流的区别

Posted on 2017-10-25

CSS 中脱离文档流，也就是将元素从普通的布局排版中拿走，其它盒子在定位的时候，会当做脱离文档流的元素不存在而进行定位。需要注意的是：

float：其它盒子会无视这个元素，但其它盒子内的文本依然会为这个元素让出位置，环绕在周围。

absolute：其它盒子与其它盒子内的文本都会无视它。

JS 笔记：常用设计模式和组件开发

Posted on 2017-08-25

模式1：Behavior
从交互需求出发设计一组“动作行为”，在主题对象上应用和扩展这些行为。

模式2：Adapter
追对不同底层实现统一接口的外部接口，让业务层面的代码可以适应不同的底层实现。

模式3：Obsever
同步多个对象的状态变化，解耦各个对象之间的依赖。

模式4：Mediator
将命令传递给所有订阅者，与订阅者们维持依赖关系，从而让订阅者彼此之间相互独立。

模式5：Omposite
对已有组件/库/功能的组合使用，提升代码复用性的最常用手段。

模式6：Partial Application
Function.prototype.bind，部分应用/固化参数

模式7：Decorate
在原有 API 基础上增加额外的功能，不改变底层库的情况下扩展功能的一种模式。

模式8：Proxy
保持原有 API 功能不变，改变执行过程和结果，不改变底层库的情况下获得不同执行结果的手段。

模式9：Aspect
对代码执行的过程进行监测和修改，不改变代码主体执行流程的情况下控制执行过程。
（preventDefault()、Symblos…）

Decorate、Proxy 和 Aspect 的联系
Decorate 强调的是声明和定义。
Proxy 强调的是功能。
Aspect 强调的是运行过程。
动态语言可以用 Decorate 来实现 Proxy 和 Aspect。

模式10：Lazy load
一些加载耗时的资源进行分段、异步加载，提升前端性能，减少相应用户的时间，某些情况下减少网页流量和带宽消耗。

总结
我们所做的一切都是为了提升抽象程度和灵活性，从而改善我们的代码质量，降低维护成本。
