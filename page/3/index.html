CSS 笔记：布局

Posted on 2017-08-18

布局方法
float, position, inline-block, table, flex, grid

水平居中
行级元素：text-align: center
块级元素：margin: auto

max-width
大部分图文内容的网页，布局都可以用 max-width 加 margin: auto。

垂直居中
单行文字：line-height
行级盒子：vertical-align: middle
绝对定位：top: 50%; left: 50%

position
用 absolute 和 margin 进行两栏布局。

main { position: relative; }
article { margin-left: 10em; }
aside { position: absolute; width: 10em; top: 0; left: 0; }

table
用 display 模拟表格布局，宽度自适应。不要使用表格标签布局。

main { display: table; }
article { display: table-cell; }
aside { display: table-cell; }

可用于实现居中效果。
.container { display: table-cell; text-align: center; vertical-align: middle; }

display: flex
将元素变为 Flexbox。

flex-direction
子元素排列方向，取值为 row、row-reverse、column、column-reverse。

flex-grow
元素扩展（占剩余空间）的能力，默认值为 0。

flex-shrink
元素收缩的能力，默认值为 1。

flex-wrap
元素在主轴方向排放时，能否换行，取值为 nowrap、wrap、wrap-reverse。

justify-content
子元素沿主轴方向的摆放，取值为 flex-start、flex-end、center、space-between、space-around。

align-items
子元素在侧轴方向的对齐方式，取值为 flex-start、flex-end、center、baseline、stretch。

align-self
对单独的元素设置对齐方式。

align-content
多行内容在容器内侧轴方向的对齐，取值为 flex-start、flex-end、center、space-between、space-around、stretch。

order
指定摆放时的顺序，默认值为 0。

strut
p 标签中的 img 图片距离底部会有一点距离，原因是文字的 baseline 在 text-bottom 之上。

lists style
list-item 会生成 Principle Block 和 Marker Box（小圆点）。
Marker Box 的内容和位置可以通过 list-style 系列属性指定。

list-style-type
none, disc, circle, square, decimal, lower-roman, upper-roman 等。

list-style-image
li { list-style-image: url(https://xxx.com/xxx/xxx.svg }

background-size
contain：保持比例，包含完整图片。
cover：保持比例，图片填满容器。

多重阴影
一个盒子可以写无限多个阴影（用来复原盒子，画logo、多啦A梦等等）。

CSS 笔记：定位与堆叠

Posted on 2017-08-18

position
static：非定位，默认值
realative：相对定位（相对自己）
absolute：绝对定位，相对非 static 祖先元素定位
fixed：相对于视口绝对定位

line-height
只有一行文字需要居中，可以不设置 height，只设置 line-height。
.box { width: 200px; line-height: 200px; text-align: center; }

z-index 堆叠层次
初始值为 auto，可以为负整数、0、正整数，数值越大，越靠近用户。

堆叠上下文的生成
Root 元素
z-index 值不为 auto 的定位元素。
设置了某些 CSS3 属性的元素，比如 opacity、transform、animation 等。

CSS 笔记：盒模型

Posted on 2017-08-18

margin 折叠
相同上下文内，margin 会合并。

margin 负值
盒子之间可以出现重叠。

box-sizing
改变盒模型计算方式，取值为：border-box, content-box。
可以用显示继承灵活控制 box-sizing 使用范围：

* { box-sizing: inherit; }
html { box-sizing: border-box; }
div { box-sizing: content-box; }

Viewport
浏览器的可视区域

行级盒
行级盒的 margin-top 和 margin-bottom 不会产生效果。
行级盒的 padding-top 和 padding-top 不影响布局。

Generated Content
控制元素，可以用于添加图标。

<i class="fa fa-comments'></i>
<style>
    .fa-comments:before { content: '\f086'; }
</style>
/*html 是表述内容和结构，我认为图标不是内容，只是修饰，所以用伪元素控制*/

CSS 笔记：层叠和单位

Posted on 2017-08-18

属性覆盖
优先级高的属性覆盖优先级低的属性， 减少了代码量。

<button class="btn">普通按钮</button>
<button class="btn btn-primary">主要按钮</button>
<style>
    .btn {
        display: inline-block;
        padding: .36em .8em;
        margin-right: .5em;
        line-height: 1.5;
        text-align: center;
        cursor: pointer;
        border-radius: .3em;
        border: none;
        background: #e6e6e6;
        color: #333;
    }
    .btn.btn-primary {
        color: #fff;
        background: #218de6;
    }
</style>

相对单位
em：相对于该元素的 font-size
rem：相对于根元素（html）的 font-size
vh：浏览器窗口高度的1%
vw：浏览器窗口宽度的1%
vmin：vh 和 vw 中的较小者
vmax：vh 和 vw 中的较大者

颜色
关键字：blue, lightsteelblue(147种)
Hex：#0000ff
RGB & RGBA：rgb(0,0,255), rgba(0,0,0,0.5)
HSL & HSLA：hsl(0,50%,50%), hsla(300, 50%, 50%, 0.5)
HSL 用于形成同一色系，如按钮悬浮时加深颜色。

CSS 笔记：浮动

Posted on 2017-08-18

定位模式
常规流（Normal Flow）
浮动（Float）
绝对定位（Absolute Positioning）

浮动
浮动元素会从常规流中脱离，漂到容器左边缘或右边缘，挤开其它元素。

clear
指定元素哪一边不能与之前的浮动框相邻，取值为 left、right、both。

clearfix
在浮动元素父级之后加一个匿名 div 清除浮动。

.clearfix::after {
    content: ' ';
    display: block;
    clear: both;
    height: 0;
    overflow: hidden;
}

块级元素上下文（BFC）
BFC 可以通过 overflow: hidden 等方法创建。
BFC 可以清除浮动。
section { overflow: hidden }
BFC 可以实现双栏布局。

img { float: left }
p { overflow: hidden; padding-left: 10px }

CSS 笔记：文本样式

Posted on 2017-08-18

通用字体族
Serif 衬线体：Georgia、宋体
Sans-Serif 无衬线体：Arial、Helvetica、黑体、微软雅黑
Cursive 手写体：Caflisch Script、楷体
Fantasy 梦幻字体：Comic Sans MS
Monospace 等宽字体：Consolas、Courier、中文字体
h1 { font-family: Helvetica, sans-serif; } /* 没有 Helvetica 时，系统根据通用字体族匹配一个字体 */

font-family 使用
英文字体放在中文字体前面。（因为中文字体包含英文，会把英文设置成比如微软雅黑）
最后总是添加通用字体族。

font-size
绝对值：xx-small, x-small, small, medium, large, x-large, xx-large
相对值：larger, smaller
百分数：相对于父元素计算值
长度：px, em
初始值为 medium（由浏览器设置决定，一般 16px）

em
元素的 font-size 属性是相对于父元素的 font-size 的计算值。
其它属性是相对于元素本身的 font-size 的计算值。
h2 { font-size: 1.5em; margin-bottom: 1em; }

line-height
段落文字一般取值 1.4~1.8
p { line-height: 1.5 }

font 缩写
h1 { font: bold 14px/1.7 Helvetica, sans-serif; } /* 斜体 粗细 大小/行高 字体族 */

Web Fonts

@font-face {
    font-family: 'Lobster';
    src: local('Lobster'),
         local('Lobster-Regular'),
         url(//lib.baomitu.com/fonts/lobster/lobster-v18-latin-regular.woff2) format('woff2'),
         url(//lib.baomitu.com/fonts/lobster/lobster-v18-latin-regular.woff') format('woff');
}
h1 { font-familt: 'Lobster', cursive; }

Web fonts 链接
Google Fonts
Baomitu CDN
Font Spider

text-align
justify 是两端对齐，但最后一行/单独一行不会起作用。
p { text-align: justify; }

white-space
指定空白符如何处理，取值为 normal、nowrap、pre 等。

word-break
指定是否允许再单词中间换行，取值为 normal、break-all、break-word 等。

CSS 笔记：概念和简单选择器

Posted on 2017-08-17

CSS 语法

使用 CSS
链接，可以被浏览器缓存
<link rel="stylesheet" href="/xxx/style.css">
嵌入，省掉一个 http 请求

<style type="text/css">
    p { color: #fff; }
</style>

内联，一般不用
<p style="color:#fff">Content</p>

简单选择器
通配选择器：* { color: #fff; }
标签选择器：p { color: #fff; }
ID 选择器：#example { color: #fff; }
类选择器：.icon { color: #fff; }

其它选择器
属性选择器：input[type="password] { color: red; }
伪类选择器：a:hover { color:red }

选择器组合
直接组合 p.warning { color: red }
后代组合 article p { color: #fff }
子代组合 article>p { color: #fff }

选择器组
body, h1, h2, h3, h4, h5, h6, ul, ol, li { margin: 0; padding: 0; }

HTML 笔记：meta 标签

Posted on 2017-08-03

介绍

<meta> 元素可提供有关页面的元信息（meta-information）

作用

编码
<meta charet = "UTF-8">

指定 HTTP Header
<meta http-equiv="Content-Security-Policy" content="script-src 'self'">

SEO 搜索引擎优化
<meta name="keywords" content="关键词">
<meta name="description" content="页面介绍">

移动设备 Viewport
<meta name="viewport" content="initial-scale=1">

关闭 iOS 电话号码自动识别
<meta name="format-detection" content="telphone=no">

360 浏览器指定内核
<meta name="render" content="webkit">

制定 IE 渲染模式
<meta http-equiv="X-UA-Compatible" content="IE=edge">

HTML 笔记：表单

Posted on 2017-08-03

GET
语义：从服务器拿数据，不会修改服务器，每次操作结果相同，会缓存
传输：数据在 url 中，一次发送

POST
语义：向服务器提交数据，会修改服务器，每次操作结果不同，不会缓存
传输：数据在 http 的请求体中，两次发送

HTTP method
GET, POST, HEAD, PUT, DELETE, OPTIONS

单行文本框
name 必须有，是向服务器发送数据对应的 key
value 是默认值
placeholder 是占位符
autofocus 在打开页面后获得焦点，比如搜索引擎输入框

input 类型
text, password, radio, checkbox, submit, file, number, range, date, month, color

非 input 类型
textarea, select

button 类型
submit, button, reset

label
input 与文字关联，提升可用性，方便盲人输入
