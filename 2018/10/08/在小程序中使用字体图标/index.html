在小程序中使用字体图标

Posted on 2018-10-08

准备 svg 格式的图标
可以到各种图标网站下载，或者自己用 Adobe Illustrator 画。

使用 IcoMoon 制作字体
进入 IcoMoon 网站，IcoMoon App -> New Project(not necessary) -> Import Icons -> Generate Font -> Download，完成下载。

引入样式文件
将下载的 style.css 文件引入项目。

字体转 base64 格式（针对小程序）
准备好下载的 fonts/xxx.woff 字体文件，进入 trasfonter 网站，Add fonts -> .woff -> Base64 encode -> Convert -> Download，完成下载。

替换字体代码（针对小程序）
用下载的 stylesheet.css 文件代码，替换 style.css 文件中 @font-face 代码。并把 [class^="icon-"], [class*=" icon-"] 改为 .icon。

使用字体图标
以类似 <span class="icon icon-home"></span> 的方式使用字体图标，设置字体大小。
